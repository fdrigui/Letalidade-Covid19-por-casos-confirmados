CFR - Letalidade de COVID-19 por caso. Análise do indicador
==============================
O Projeto consiste na analise do indicador CFR - Taxa de letalidade do COVID-19 por casos confirmados.<br>
Quais são os problemas em tentar estimar a taxa de mortalidade com esse indicador e como o contexto influencia no resultado final? para saber mais sobre o assunto, [Clique aqui](./reports/draft.md)
